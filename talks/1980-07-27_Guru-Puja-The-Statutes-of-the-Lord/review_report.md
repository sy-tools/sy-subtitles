# Language Review – 1980-07-27_Guru-Puja-The-Statutes-of-the-Lord, 2026-07-14

## Process

Review of `transcript_uk.txt` (full paragraphed Ukrainian text) against
`transcript_en.txt`, using 2 parallel reviewers (L = Language, S = SY Domain)
+ 1 Critic filter, per `templates/language_review_template.md`,
`glossary/CLAUDE.md`, `glossary/terms_lookup.yaml`, `glossary/terms_context.yaml`.
Paragraph numbers are 1-based line numbers in `transcript_uk.txt`
(header = lines 1–6, body = lines 7–69).

This is **round 2**. It supersedes the round of 2026-05-29 (see git history),
which fixed systematic deity-pronoun capitalization (~80 tokens of Шрі Матаджі's
Я/Мене/Мені/Моя/Сама). All round-1 decisions were re-verified and left intact,
with one documented reversal: round 1's Critic declined the dash normalization
as "a sweeping style change"; round 2 applies it, because the rule in
`glossary/CLAUDE.md` explicitly prescribes ` – ` (U+2013) and the rest of the
corpus (40+ talks) follows it — this file was one of only three outliers.
