| L | L1 | **Keep** | Clear grammatical error: «які вони» creates a double subject; English original «how they were so beautiful» is a content clause requiring «як», not the relative pronoun «які». |
| L | L2 | **Keep** | «такі дуже» mixes adjectival demonstrative «такі» with adverb «дуже»; standard Ukrainian uses one or the other. Minimal fix preserves «такі» emphasis. |
| L | L3 | **Remove** | Both «було двоє» (impersonal) and «були двоє» (plural agreement) are attested in modern Ukrainian. Not a clear error – stylistic preference. |
| L | L4 | **Keep** | «робіння» is not in СУМ / standard dictionaries; the deverbal noun for «робити» is «роблення». Clear lexical fix. |
| L | L5 | **Remove** | The English original «They're all right» is itself informal/elliptical. In colloquial Ukrainian, «Вони нормально» with elided verb («танцюють») is acceptable. Style preference. |
| L | L6 | **Remove** | English original «and a short person» is informal/quirky. While «короткий» for human height is non-standard, the translator's choice mirrors the informal English; not an unambiguous error. |
| L | L7 | **Keep** | Glossary specifies «Муладхара» (nominative). The form «Муладхар» drops the -а; the very next sentence uses «Муладхарою» (instr. of «Муладхара»), confirming the intended form. |

| L | L10 | **Remove** | «Як результат» as a sentence-initial calque is increasingly accepted in modern Ukrainian. Style preference, not error. |
